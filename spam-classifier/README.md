# Spam Email Classifier

## Overview
This project detects whether an email is spam or not using Machine Learning.

## Tech Stack
- Python
- Scikit-learn
- TF-IDF
- Naive Bayes

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py

## Output
- Shows accuracy
- Predicts custom message

## Concepts Used
- NLP (TF-IDF)
- Classification (Naive Bayes)
